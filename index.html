My Work

Pudding Power

Skills and Resources: UX and UI Design, using Sketch to create mock ups
Project Description: Pudding Power is a responsive web app targeted at users that wish to find and save plant based dessert recipes, to recreate at home or with friends! The UX design process was followed end-to-end, from initial user research, development of personas and user flows, through to narrowing down of the requirements and stories, creation of low, mid and high fidelity designs and mock ups.

Feline Friends

Skills and Resources: The User Centred Design process was followed; from discovery, to concepting to prototyping and user testing. UX and UI design disciplines were utilised using Balsamiq and Sketch as design tools.
Project Description: Feline Friends is an online retail product, targeting cat owners. This would be a ‘one stop shop’, allowing cat owners to find varied brands and products as well as nearby specialist services, all using their device. Wherever and whenever they need to!
